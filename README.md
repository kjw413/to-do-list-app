# 업무 To-Do List

메모장 대신 쓰는 가벼운 업무 관리 데스크톱 앱입니다.
**우선순위 · 마감일/남은기간(영업일)/긴급도 · 카테고리**로 잔여 업무의 급한 정도를 한눈에 봅니다.
**캘린더**에 휴가·출장·교육 일정을 등록하면 그 기간(과 주말·공휴일)을 **비근무일**로 빼고 남은기간을 다시 계산합니다.

Python 표준 라이브러리(Tkinter)만 사용하므로 추가 설치가 필요 없고,
단일 실행파일(`.exe`)로 빌드해 작업표시줄에 고정해 쓸 수 있습니다.

---

## 빠르게 실행하기 (빌드 없이)

```
run.bat  더블클릭
```
또는 터미널에서:
```
py todo_app.py
```

## 실행파일(.exe)로 만들기 — 설치형으로 쓰기

```
build_exe.bat  더블클릭
```
- PyInstaller가 없으면 자동 설치 후 빌드합니다(1~2분).
- 결과물: `dist\TodoList.exe`
- 이 `.exe` 하나만 있으면 Python 없이도 다른 PC에서 실행됩니다.
- 바탕화면이나 작업표시줄에 고정해 두고 메모장처럼 바로 여세요.

## 사용법

| 기능 | 방법 |
|------|------|
| 빠른 추가 | 상단 입력칸에 내용 입력 후 **Enter** (우선순위 '중간', 마감일 없음으로 등록) |
| 상세 추가 | **상세 추가…** 버튼 → 카테고리·우선순위·마감일·메모 입력 |
| 수정 | 항목 **더블클릭** (또는 선택 후 Enter) |
| 완료/취소 | **Space**, 또는 '완료' 열 더블클릭, 또는 하단 **완료/취소** 버튼 |
| 삭제 | **Delete** 키 또는 **삭제** 버튼 |
| 정렬 | 상단 '정렬' 선택, 또는 표의 헤더 클릭 |
| 필터 | '카테고리' 선택 / '완료 숨기기' / '검색' |
| 오늘 요약 | **Ctrl+D** 또는 하단 **오늘 요약** 버튼 (지연·임박 업무 모아보기) |
| 캘린더 | **Ctrl+L** 또는 상단 **📅 캘린더** 버튼 (휴가·출장·교육 일정 관리) |

### 행 색상 = 긴급도
- 🔴 **빨강**: 영업일 기준 1일 전~마감일 지남
- 🟠 **주황**: 영업일 기준 2~4일 전
- 🟡 **노랑**: 영업일 기준 5~7일 전
- ⚪ **회색 취소선**: 완료

### 정렬 '긴급도순'
마감 임박/지연 정도와 우선순위를 함께 계산해, **가장 급한 일이 맨 위**로 올라옵니다.

## 캘린더 · 업무 스케줄 관리

업무 납기를 정확히 지키려면 **실제로 일할 수 있는 날**만 세어야 합니다.
이 앱은 **주말 + 대한민국 공휴일 + 내가 등록한 일정(휴가·출장·교육)** 을 모두 **비근무일**로 보고
모든 업무의 **남은기간(영업일)·긴급도·색상·오늘 요약**을 다시 계산합니다.

### 사용법
- 상단 **📅 캘린더** 버튼 또는 **Ctrl+L** 로 월간 달력을 엽니다.
- **날짜를 클릭**하면 그 날짜로 새 일정 추가 창이 열립니다. (또는 **＋ 일정 추가**)
- 일정에는 **종류(휴가/출장/교육/기타) · 제목 · 시작일 · 종료일 · 메모**를 넣습니다.
  종료일을 비우면 하루짜리 일정입니다.
- **◀ 이전달 / 오늘 / 다음달 ▶** 으로 달을 이동하고, 아래 **이번 달 일정** 표에서 수정·삭제합니다.

### 달력 색상
- 🟦 휴가 · 🟧 출장 · 🟪 교육 (일정이 있는 날)
- 🔴 일요일/공휴일 · 🔵 토요일 · 🟨 오늘
- 셀 안에는 `※공휴일`, `[종류] 일정명`, `📌마감 N건`(그날 마감인 미완료 업무 수)이 표시됩니다.

### 비근무일 처리 끄기
출장이지만 그날 업무가 가능한 경우 등, 비근무일로 빼고 싶지 않은 일정은
추가/수정 창에서 **"비근무일로 처리"** 체크를 해제하면 됩니다.
이 일정은 달력에는 보이되 남은기간 계산에는 영향을 주지 않습니다.

### 공휴일 갱신 (중요)
공휴일은 프로그램 폴더의 **`DB_holiday.xlsx`** 파일을 읽어 비근무일로 처리합니다.

| 열 | 내용 | 예시 |
|----|------|------|
| A | 날짜 | `2026-12-25` |
| B | 요일 | 금 |
| C | 공휴일명 | 크리스마스 |

- 첫 행은 머리글로 보고 건너뜁니다. **A열(날짜)·C열(이름)** 만 사용합니다.
- 다음 해 공휴일·대체공휴일은 이 엑셀에 행을 추가하면 됩니다(코드 수정 불필요).
- **📅 캘린더**를 다시 열면 엑셀 변경분이 즉시 반영됩니다(앱 재시작 불필요).
- 엑셀을 읽으려면 `openpyxl` 이 필요합니다. 파일이 없거나 `openpyxl` 미설치 시
  코드 내장 목록(`PUBLIC_HOLIDAYS_RAW`, 2025~2027년)으로 자동 대체됩니다.
- `.exe` 로 빌드해 쓸 때는 **`DB_holiday.xlsx` 를 `TodoList.exe` 와 같은 폴더**에 두세요.

## 알림 · 자동 실행

### 시작 시 '오늘 요약' 알림
앱을 열면 **지연·1영업일 이내 마감** 업무를 팝업으로 모아 보여줍니다(급한 일이 없으면 표시하지 않음).
- 언제든 **Ctrl+D** 또는 **오늘 요약** 버튼으로 다시 볼 수 있습니다.
- 팝업 하단 또는 **설정** 메뉴에서 *시작 시 표시*를 끌 수 있습니다.
- 앱을 켜둔 채 자정을 넘겨도 남은기간이 자동으로 갱신됩니다(10분 주기).

### Windows 시작 시 자동 실행
**설정 → Windows 시작 시 자동 실행** 을 켜면 로그인할 때 자동으로 실행됩니다.
- 관리자 권한·추가 설치 불필요 (현재 사용자 레지스트리 `HKCU\...\Run`에 등록).
- 끄면 즉시 해제됩니다.
- exe로 실행 중이면 그 exe 경로가 등록되므로, **`TodoList.exe` 위치를 옮기면 자동 실행을 한 번 껐다 켜** 주세요.

### 스티커 모드 · 항상 위에 표시
화면 한쪽에 띄워두는 **스티커 메모처럼** 쓰고 싶을 때:
- **설정 → 간단히 보기(스티커 모드)**: 필터줄·하단 버튼과 부가 열(우선순위·카테고리·마감일)을 숨겨 `완료·업무·남은기간`만 남기고, 창을 **폭 280px까지** 좁힐 수 있습니다(긴급도 색상은 그대로 유지).
- **설정 → 항상 위에 표시**: 다른 창 위에 항상 떠 있게 합니다.
- 스티커 모드에서도 추가는 상단 입력칸, 완료/삭제는 단축키(Space/Delete)로 가능합니다.

## 데이터 저장 위치

```
%APPDATA%\TodoList\tasks.json       ← 업무
%APPDATA%\TodoList\schedules.json   ← 캘린더 일정(휴가/출장/교육)
```
- 업무·일정을 추가/수정/삭제할 때마다 **자동 저장**됩니다.
- 메뉴 **파일 → 데이터 폴더 열기** 로 바로 찾아갈 수 있습니다.
- 백업: **파일 → 백업(내보내기)**, 복원/이전: **파일 → 가져오기(병합)**
